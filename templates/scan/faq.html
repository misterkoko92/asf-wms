    {% trans "Vue d'ensemble" %}

    {% blocktrans trimmed %} Cette page décrit le fonctionnement complet du WMS Scan, du stock à l'expédition. Les flux sont conçus pour être rapides sur mobile et pour rester cohérents en base. {% endblocktrans %}

    • {% blocktrans trimmed %}Produit : fiche article (SKU, nom, marque, poids, emplacement).{% endblocktrans %}
    • {% blocktrans trimmed %}Lot : stock réel par produit (quantité, péremption, lot, emplacement).{% endblocktrans %}
    • {% blocktrans trimmed %}Carton : unité de préparation (DRAFT, PICKING, PACKED, ASSIGNED, LABELED, SHIPPED).{% endblocktrans %}
    • {% blocktrans trimmed %}Expédition : regroupe des cartons pour une destination.{% endblocktrans %}
    • {% blocktrans trimmed %}Suivi expédition : journal des scans (QR) avec horodatage.{% endblocktrans %}

    {% trans "Statuts cle" %}

    • {% blocktrans trimmed %}Carton : Créé (draft), En préparation (picking), Prêt (packed), Affecté (assigned), Étiqueté (labeled), Expédié (shipped).{% endblocktrans %}
    • {% blocktrans trimmed %}Expédition : Création (draft), En cours (picking), Prêt (packed), Planifié (planned), Expédié (shipped), Reçu escale (received_correspondent), Livré (delivered).{% endblocktrans %}
    • {% blocktrans trimmed %}Litige : drapeau transverse sur expédition (is_disputed), activable à tout moment du suivi.{% endblocktrans %}
    • {% blocktrans trimmed %}Réception : DRAFT tant que les lignes ne sont pas réceptionnées.{% endblocktrans %}

    {% trans "Flux principaux" %}

    {% trans "Les parcours ci-dessous résument l'ordre des pages à ouvrir et des boutons principaux à utiliser." %}

    {% trans "Réception de don et mise en stock" %}

    {% blocktrans trimmed %} 1. Utiliser Réception palette pour créer le dossier logistique ou Réception standard (don/palette) pour saisir directement des lignes de stock. 2. Renseigner le donateur, le transporteur et la date. 3. Importer un listing ou saisir les produits manuellement. 4. Utiliser Réceptionner maintenant ou la validation du listing pour créer les lots. 5. Contrôler le résultat dans Vue Réception puis Vue Stock. {% endblocktrans %}

    {% trans "Mise à jour des stocks" %}

    {% blocktrans trimmed %} 1. Ouvrir MAJ stock. 2. Identifier le produit par recherche ou avec le bouton Scan. 3. Saisir quantité, lot et péremption. 4. Enregistrer pour créer un nouveau lot sur l'emplacement par défaut. 5. Vérifier la nouvelle quantité dans Vue Stock. {% endblocktrans %}

    {% trans "Créer des colis" %}

    {% blocktrans trimmed %} 1. Utiliser Préparer des colis pour des produits unitaires ou Préparer des kits pour des préparations répétées. 2. Choisir le format, le kit ou les produits. 3. Ajouter les lignes avec Ajouter un produit ou Ajouter. 4. Lancer la préparation avec le bouton principal de la page. 5. Vérifier le résultat dans Vue Colis. {% endblocktrans %}

    {% trans "Créer une expédition avec des colis déjà préparés" %}

    {% blocktrans trimmed %} 1. Ouvrir Créer une expédition. 2. Choisir la destination puis les contacts proposés. 3. Ajouter les colis Prêt non affectés. 4. Utiliser Enregistrer un brouillon si le dossier n'est pas finalisé ou Créer Expédition pour générer la référence finale. 5. Étiqueter les colis puis suivre l'avancement dans Vue Expéditions et Suivi expéditions. {% endblocktrans %}

    {% trans "Créer une expédition sans colis préparés" %}

    {% blocktrans trimmed %} 1. Ouvrir Créer une expédition et renseigner destination, expéditeur, destinataire et correspondant. 2. Utiliser Enregistrer un brouillon pour figer le dossier ou Créer des colis multi-produits pour basculer vers la préparation. 3. Préparer ensuite les colis depuis les produits ou les kits. 4. Revenir sur l'expédition pour affecter, étiqueter et suivre les colis. {% endblocktrans %}

    {% trans "Commande association sur le portal" %}

    {% blocktrans trimmed %} 1. Se connecter au Portal association. 2. Ouvrir Nouvelle commande. 3. Choisir destination, destinataire, quantités et documents éventuels. 4. Envoyer la commande. 5. Suivre la validation ASF depuis Mes commandes puis récupérer les documents sur la fiche commande. {% endblocktrans %}

    {% trans "Planning hebdomadaire" %}

    {% blocktrans trimmed %} 1. Créer un run depuis Planning > Nouveau run. 2. Ouvrir le run puis lancer Générer le planning. 3. Vérifier les issues et les versions. 4. Ajuster les affectations dans la version courante. 5. Publier la version, exporter le fichier et mettre à jour les expéditions si nécessaire. {% endblocktrans %}

    • {% blocktrans trimmed %}Voir les états regroupe les pages de consultation: Vue Stock, Vue Kits, Vue Colis, Vue Expéditions, Vue Commande et Vue Réception. Leur objectif est de contrôler l'état courant et d'utiliser les actions rapides de tableau.{% endblocktrans %}
